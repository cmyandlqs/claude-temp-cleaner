# 需求文档（PRD）- Claude Temp Cleaner（Windows + PySide6）

## 1. 背景与目标
Claude 会在工作目录中生成 `tmpclaude-` 前缀的临时文件/目录，影响磁盘空间与工程整洁。
目标是提供一个简洁、美观的 Windows 桌面工具，快速扫描并清理这些临时项，默认安全（预览 + 确认）。

## 2. 目标用户
本地开发者 / 设计师 / 内容创作者等 Windows 用户。

## 3. 核心需求
- R1：用户启动后可选择一个文件夹作为扫描根目录。
- R2：选择后立即递归扫描，匹配名称以 `tmpclaude-` 开头的文件与目录。
- R3：扫描结果列表展示（路径、类型、大小、最后修改时间）。
- R4：默认仅预览；需明确确认后再删除。
- R5：删除后显示统计信息（删除数量、总大小、耗时、失败项）。
- R6：整体运行快、占用低、启动轻量。

## 4. 非功能需求
- 性能：对大目录扫描响应快，避免卡 UI。
- 安全：二次确认；明确显示将删除的项目。
- 体验：界面简洁、偏 mac 风格（圆角、浅色、弱阴影、柔和蓝色强调）。
- 兼容：Windows 10/11。

## 5. 约束与假设
- 仅匹配 `tmpclaude-` 前缀。
- 不排除任何目录（含 .git、node_modules）。
- 只做 Windows 版。

## 6. 交互流程
1) 启动应用，显示主界面。
2) 用户选择文件夹后立即开始扫描。
3) 扫描完成后展示结果列表与统计信息。
4) 用户点击“清理”，弹出确认框。
5) 确认后执行删除，结束后显示删除结果与失败项。

## 7. 成功标准
- 能正确列出并删除目标项。
- 默认安全（预览 + 确认）。
- 执行速度快，输出清晰，体验顺滑。
